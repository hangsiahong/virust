# Virust v0.4 Implementation Plan - Hybrid Rendering with Bun

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task.

**Goal:** Add Next.js-style server-side rendering (SSR) to Virust using Bun as the JavaScript runtime, enabling React Server Components with async data fetching and client-side hydration.

**Architecture:** Virust will integrate Bun as an external JavaScript runtime process. Route handlers use `#[render_component]` to return `RenderedHtml`, which triggers IPC communication to Bun. Bun loads and executes React components, rendering them to HTML with embedded hydration data. The browser receives HTML and hydrates with React for interactivity.

**Tech Stack:** Rust (Axum, tokio), Bun (JavaScript runtime), React (SSR), IPC (stdin/stdout JSON), Filesystem routing

use std::collections::HashMap;
use std::path::{Path, PathBuf};
use anyhow::Result;

pub struct ComponentRegistry {
    components: HashMap<String, PathBuf>,
}

impl ComponentRegistry {
    pub fn new() -> Self {
        Self {
            components: HashMap::new(),
        }
    }

    pub fn discover(&mut self, web_dir: &Path) -> Result<()> {
        let components_dir = web_dir.join("components");
        if !components_dir.exists() {
            return Ok(());
        }

        self.scan_directory(&components_dir)?;
        Ok(())
    }

    fn scan_directory(&mut self, dir: &Path) -> Result<()> {
        use std::fs;

        for entry in fs::read_dir(dir)? {
            let entry = entry?;
            let path = entry.path();

            if path.is_dir() {
                self.scan_directory(&path)?;
            } else {
                if let Some(ext) = path.extension() {
                    if ext == "jsx" || ext == "js" || ext == "tsx" || ext == "ts" {
                        if let Some(name) = self.extract_component_name(&path) {
                            self.components.insert(name, path);
                        }
                    }
                }
            }
        }
        Ok(())
    }

    fn extract_component_name(&self, path: &Path) -> Option<String> {
        path.file_stem()
            .map(|s| s.to_string_lossy().to_string())
    }

    pub fn get(&self, name: &str) -> Option<&PathBuf> {
        self.components.get(name)
    }

    pub fn list(&self) -> Vec<&str> {
        self.components.keys().map(|s| s.as_str()).collect()
    }
}
```

use std::process::{Child, Command, Stdio};
use std::io::{BufRead, BufReader, Write};
use anyhow::Result;
use serde_json::Value;

use crate::component::{ComponentRegistry, RenderedOutput};

pub struct BunRenderer {
    bun_process: Child,
    component_registry: ComponentRegistry,
    stdin: Option<std::process::ChildStdin>,
    stdout: Option<BufReader<std::process::ChildStdout>>,
}

impl BunRenderer {
    pub fn new() -> Result<Self> {
        let component_registry = ComponentRegistry::new();

        // Spawn Bun process
        let mut child = Command::new("bun")
            .arg("run")
            .arg("/opt/virust/bun/renderer.js")  // Will be created in Task 6
            .stdin(Stdio::piped())
            .stdout(Stdio::piped())
            .stderr(Stdio::inherit())
            .spawn()
            .map_err(|e| anyhow::anyhow!("Failed to spawn Bun: {}. Is Bun installed?", e))?;

        let stdin = child.stdin.take();
        let stdout = child.stdout.take().map(BufReader::new);

        Ok(Self {
            bun_process: child,
            component_registry,
            stdin,
            stdout,
        })
    }

    pub fn set_web_dir(&mut self, web_dir: &std::path::Path) -> Result<()> {
        self.component_registry.discover(web_dir)
    }

    pub fn component_count(&self) -> usize {
        self.component_registry.list().len()
    }
}
```

    fn send_request(&mut self, request: &Value) -> Result<Value> {
        let stdin = self.stdin.as_mut().ok_or_else(|| anyhow::anyhow!("stdin not available"))?;

        let request_str = serde_json::to_string(request)?;
        writeln!(stdin, "{}", request_str)?;
        stdin.flush()?;

        Ok(())
    }

    fn receive_response(&mut self) -> Result<Value> {
        let stdout = self.stdout.as_mut().ok_or_else(|| anyhow::anyhow!("stdout not available"))?;

        let mut line = String::new();
        stdout.read_line(&mut line)?;

        if line.is_empty() {
            return Err(anyhow::anyhow!("Empty response from Bun"));
        }

        let response: Value = serde_json::from_str(&line.trim())?;
        Ok(response)
    }
}

impl BunRenderer {
    // ... existing code ...

    pub async fn render_component(&mut self, name: &str, props: Value) -> Result<RenderedOutput> {
        // Find component path
        let component_path = self.component_registry.get(name)
            .ok_or_else(|| anyhow::anyhow!("Component not found: {}", name))?;

        let request = serde_json::json!({
            "type": "render",
            "component": component_path.to_string_lossy(),
            "props": props
        });

        self.send_request(&request)?;
        let response = self.receive_response()?;

        if let Some(error) = response.get("error") {
            return Err(anyhow::anyhow!("Component render error: {}", error));
        }

        let html = response["html"].as_str()
            .ok_or_else(|| anyhow::anyhow!("Response missing 'html' field"))?
            .to_string();

        let hydration_data = response["hydrationData"].as_str()
            .ok_or_else(|| anyhow::anyhow!("Response missing 'hydrationData' field"))?
            .to_string();

        Ok(RenderedOutput::new(html, hydration_data))
    }
}

pub struct BunSupervisor {
    renderer: Option<BunRenderer>,
}

impl BunSupervisor {
    pub fn new() -> Self {
        Self {
            renderer: None,
        }
    }

    pub async fn ensure_running(&mut self) -> Result<&mut BunRenderer> {
        if self.renderer.is_none() || !self.renderer.as_ref().unwrap().is_alive() {
            self.renderer = Some(BunRenderer::new()?);
        }
        Ok(self.renderer.as_mut().unwrap())
    }

    pub fn shutdown(mut self) -> Result<()> {
        if let Some(renderer) = self.renderer.take() {
            drop(renderer);
        }
        Ok(())
    }
}
```

    let input = parse_macro_input!(item as ItemFn);
    let args = parse_macro_input!(attr with AttributeArgs);

    // Parse component name from attribute
    let component_name = args.iter()
        .filter_map(|arg| {
            if let syn::NestedMeta::Lit(syn::Lit::Str(lit)) = arg {
                Some(lit.value())
            } else {
                None
            }
        })
        .next()
        .expect("#[render_component(\"ComponentName\")] requires a component name");

    let fn_name = &input.sig.ident;
    let fn_name_str = fn_name.to_string();

    // Extract parameters
    let mut path_params = Vec::new();
    let mut body_params = Vec::new();

    for param in &input.sig.inputs {
        match param {
            syn::FnArg::Typed(pat) => {
                if let syn::Pat::Ident(pat_ident) = &*pat.pat {
                    let param_name = &pat_ident.ident;
                    let param_type = &pat.ty;

                    // Check for #[path] and #[body] attributes
                    let attrs = &pat.attrs;
                    let has_path_attr = attrs.iter().any(|a| {
                        a.path().is_ident("path")
                    });
                    let has_body_attr = attrs.iter().any(|a| {
                        a.path().is_ident("body")
                    });

                    if has_path_attr {
                        path_params.push(quote! {
                            #param_name: axum::extract::Path<#param_type>
                        });
                    } else if has_body_attr {
                        body_params.push(quote! {
                            #param_name: axum::extract::Json<#param_type>
                        });
                    }
                }
            }
            _ => {}
        }
    }

    let all_wrapper_params = quote::concat![
        &path_params,
        &body_params
    ];

    // Generate wrapper function
    let expanded = quote! {
        #input

        pub fn #fn_name##_wrapper(__virust_bun: &::virust_bun::BunRenderer) -> impl axum::response::IntoResponse {
            use ::virust_runtime::RenderedHtml;
            use ::serde_json::json;

use axum::response::{Html, IntoResponse};
use std::sync::Arc;

pub struct RenderedHtml {
    pub component_name: String,
    pub props: Value,
}

// Channel for sending render requests
lazy_static::lazy_static! {
    static ref RENDER_TX: mpsc::Sender<RenderRequest> = {
        let (tx, mut rx) = mpsc::channel(100);

        tokio::spawn(async move {
            let mut renderer_opt: Option<::virust_bun::BunRenderer> = None;

            while let Some(req) = rx.recv().await {
                // Ensure renderer is running
                if renderer_opt.is_none() || !renderer_opt.as_ref().unwrap().is_alive() {
                    if let Ok(r) = ::virust_bun::BunRenderer::new() {
                        renderer_opt = Some(r);
                    }
                }

                if let Some(ref mut renderer) = renderer_opt {
                    match renderer.render_component(&req.component_name, req.props).await {
                        Ok(output) => {
                            let _ = req.tx.send(Ok(output));
                        }
                        Err(e) => {
                            let _ = req.tx.send(Err(anyhow::anyhow!("Render failed: {}", e)));
                        }
                    }
                } else {
                    let _ = req.tx.send(Err(anyhow::anyhow!("Failed to initialize Bun")));
                }
            }
        });

        tx
    };
}

struct RenderRequest {
    component_name: String,
    props: Value,
    tx: mpsc::Sender<Result<::virust_bun::RenderedOutput, anyhow::Error>>,
}

    pub fn with_props(component_name: &str, props: Value) -> Self {
        Self {
            component_name: component_name.to_string(),
            props,
        }
    }

    pub fn new(component_name: &str) -> Self {
        Self {
            component_name: component_name.to_string(),
            props: Value::Object(Default::default()),
        }
    }

    pub async fn render(self) -> Result<String, anyhow::Error> {
        let (tx, mut rx) = mpsc::channel(1);

        RENDER_TX.send(RenderRequest {
            component_name: self.component_name,
            props: self.props,
            tx,
        }).await?;

        let output = rx.recv().await.ok_or_else(|| anyhow::anyhow!("No response"))??;

        Ok(self.wrap_html(output.html, output.hydration_data))
    }

        format!(

            body,
            hydration_data
        )
    }
}

import { renderToStaticMarkup } from 'react-dom/server';
import readline from 'readline';
import { readFileSync } from 'fs';

const RENDER_CACHE = new Map();

function isClientComponent(componentPath) {
  try {
    const source = readFileSync(componentPath, 'utf-8');
    return source.includes("'use client'") || source.includes('"use client"');
  } catch {
    return false;
  }
}

async function loadComponent(componentPath) {
  if (RENDER_CACHE.has(componentPath)) {
    return RENDER_CACHE.get(componentPath);
  }

  // Clear require cache for hot reload
  if (typeof require.cache[componentPath] !== 'undefined') {
    delete require.cache[componentPath];
  }

  const component = await import(componentPath);
  RENDER_CACHE.set(componentPath, component);
  return component;
}

const renderer = {
  async render(componentPath, props) {
    try {
      const isClient = isClientComponent(componentPath);

      if (isClient) {
        // Client component - return placeholder
        return {
          html: `<div data-component="${componentPath}" data-props='${JSON.stringify(props)}' data-client></div>`,
          hydrationData: JSON.stringify(props),
          isClientComponent: true
        };
      }

      // Server component - render normally
      const component = await loadComponent(componentPath);
      const React = await import('react');
      const html = renderToStaticMarkup(
        React.createElement(component.default, props)
      );

      return {
        html,
        hydrationData: JSON.stringify(props),
        isClientComponent: false
      };
    } catch (error) {
      return {
        error: error.message,
        stack: error.stack
      };
    }
  },

  invalidate(componentPath) {
    RENDER_CACHE.delete(componentPath);
    return { ok: true };
  }
};

// IPC loop
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout,
  terminal: false
});

rl.on('line', async (line) => {
  if (line.trim() === 'EXIT') {
    process.exit(0);
  }

  try {
    const request = JSON.parse(line);

    if (request.type === 'ping') {
      console.log(JSON.stringify({ pong: true }));
    } else if (request.type === 'render') {
      const result = await renderer.render(request.component, request.props);
      console.log(JSON.stringify(result));
    } else if (request.type === 'invalidate') {
      const result = renderer.invalidate(request.component);
      console.log(JSON.stringify(result));
    }
  } catch (error) {
    console.log(JSON.stringify({ error: error.message }));
  }
});

import React from 'react';

// Global component registry
window.VIRUST_COMPONENTS = {};

async function hydrate() {
  const rootEl = document.getElementById('root');
  if (!rootEl) return;

  // Find all client component placeholders
  const clientComponents = rootEl.querySelectorAll('[data-client]');

  for (const el of clientComponents) {
    const componentName = el.getAttribute('data-component');
    const props = JSON.parse(el.getAttribute('data-props'));

    // Load component
    if (window.VIRUST_COMPONENTS[componentName]) {
      const Component = window.VIRUST_COMPONENTS[componentName];

      // Create container
      const container = document.createElement('div');
      el.parentNode.replaceChild(container, el);

      // Hydrate
      hydrateRoot(container, React.createElement(Component, props));
    }
  }
}

// Auto-hydrate on load
if (document.readyState === 'loading') {
  document.addEventListener('DOMContentLoaded', hydrate);
} else {
  hydrate();
}

use std::path::Path;
use tokio::fs;
use anyhow::Result;

pub async fn watch_components(web_dir: &Path, mut tx: tokio::sync::mpc::Sender<ComponentChange>) -> Result<()> {
    let components_dir = web_dir.join("components");

    if !components_dir.exists() {
        return Ok(());
    }

    let mut interval = tokio::time::interval(tokio::time::Duration::from_millis(500));

    let mut last_modifications = std::collections::HashMap::new();

    loop {
        interval.tick().await;

        scan_directory(&components_dir, &mut last_modifications, &mut tx).await?;
    }
}

async fn scan_directory(
    dir: &Path,
    last_modifications: &mut std::collections::HashMap<std::path::PathBuf, std::time::SystemTime>,
    tx: &mut tokio::sync::mpc::Sender<ComponentChange>,
) -> Result<()> {
    use std::fs;

    let entries = fs::read_dir(dir).await?;

    for entry in entries {
        let entry = entry?;
        let path = entry.path();

        if path.is_dir() {
            scan_directory(&path, last_modifications, tx).await?;
            continue;
        }

        if let Some(ext) = path.extension() {
            if ext == "jsx" || ext == "js" || ext == "tsx" || ext == "ts" {
                let metadata = fs::metadata(&path).await?;
                let modified = metadata.modified()?;

                let last_modified = last_modifications.get(&path);

                if last_modified != Some(&modified) {
                    last_modifications.insert(path.clone(), modified);

                    let component_name = path.file_stem()
                        .map(|s| s.to_string_lossy().to_string())
                        .unwrap_or_default();

                    tx.send(ComponentChange {
                        component_name,
                        path: path.clone(),
                        change_type: ChangeType::Modified,
                    }).await?;
                }
            }
        }
    }

    Ok(())
}

#[derive(Debug, Clone)]
pub struct ComponentChange {
    pub component_name: String,
    pub path: std::path::PathBuf,
    pub change_type: ChangeType,
}

#[derive(Debug, Clone)]
pub enum ChangeType {
    Modified,
    Created,
    Deleted,
}
```

# Virust v0.4 Release Notes

**Release Date:** 2026-03-06

## Overview

Virust v0.4 introduces server-side rendering (SSR) with Bun, enabling Next.js-style React Server Components.

## What's New

### Server-Side Rendering

- Bun integration for rendering React components on the server
- Support for async components with server-side data fetching
- Client components with 'use client' directive
- Automatic hydration in the browser

### New Templates

- `ssr-blog` - Blog template with SSR
- `ssr-dashboard` - Admin dashboard template

### Developer Experience

- Component hot-reloading in dev mode
- Improved error messages for failed renders
- TypeScript support for components

## Migration from v0.3

SSR is opt-in. Existing v0.3 projects continue to work without changes.

To add SSR to an existing project:

1. Install Bun: `curl -fsSL https://bun.sh/install | bash`
2. Create `web/components/` directory
3. Add components with `#[render_component]` attribute
4. Run `virust dev`

## Breaking Changes

None. v0.4 is fully backward compatible with v0.3.

## Known Limitations

- Bun must be installed for SSR to work
- Client components cannot use async/await
- SSR adds ~50ms to initial page load (acceptable for most use cases)

## Next Steps

v0.5 will add static site generation (SSG) and incremental static regeneration (ISR).
